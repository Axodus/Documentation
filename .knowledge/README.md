# AXODUS KNOWLEDGE LAYER

This directory contains semantic memory and operational intelligence packs for AI agents and development systems.

These files are not public-facing documentation.

They are designed to provide:

- ecosystem semantics
- architectural reasoning
- governance invariants
- operational relationships
- runtime assumptions
- terminology consistency
- retrieval-oriented context

Public documentation should derive from these knowledge packs whenever possible.

Knowledge packs are optimized for:

- AI retrieval
- semantic indexing
- operational reasoning
- documentation generation
- governance support systems
- ACS runtime coordination

The `.knowledge` layer acts as the semantic infrastructure of the Axodus ecosystem.
