## Purpose

Lottery participation can create financial and behavioral risk. CryptoDraw documentation must encourage responsible participation and avoid manipulative language.

## Required Themes

Participation does not guarantee any prize or profit. Lottery outcomes are random, and odds should be understood before participation. Availability may depend on location, age, and applicable rules. Users should only participate with amounts they are prepared to lose. Claim windows and deadlines apply.

If crypto assets are used, their value may change. Users remain responsible for wallet transactions, network fees, addresses, and transaction review.

## User Safety Controls

Possible controls include spending limits, cooldown periods, maximum tickets per draw, self-exclusion if implemented, region restrictions if required, age confirmation if required, risk acknowledgement, educational prompts, probability explainers, and support resources where applicable.

## Prohibited Messaging

Public content must not use guaranteed winning, easy money, passive income, risk-free lottery, investment lottery, play to get rich, recover losses, urgency pressure, misleading odds, or hidden cost language.
