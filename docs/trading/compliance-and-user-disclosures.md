## Purpose

Trading products require clear user disclosures. The purpose is to make risk, responsibility, and limitations explicit.

This page is not legal advice and does not claim legal or regulatory compliance.

## Core Disclosures

- Axodus Trading documentation and tools are for information, education, automation support, or product usage and are not personalized financial advice.
- No strategy, bot, signal, or trading tool can guarantee profit.
- Users are responsible for their own capital, exchange accounts, wallets, API keys, risk settings, leverage, and decisions.
- Markets are volatile and strategies can lose money.
- Futures and leveraged products can amplify losses and cause liquidation.
- Exchanges can experience outages, API failures, restrictions, or account issues.
- API keys must be protected, limited, and revocable. Withdrawal permissions should be disabled.
- Backtests and historical results do not guarantee future performance.
- User-facing CEX API strategies are intended to run in user-owned accounts unless otherwise documented.
- Strategies may be paused, deprecated, restricted, or changed due to market, risk, technical, governance, or security reasons.

## Required Disclosures by Product Type

Signal-only tools require no-financial-advice, no-profit-guarantee, user decision responsibility, and market risk language.

CEX API bots require user responsibility, API key risk, exchange risk, strategy failure risk, and pause or revocation guidance.

Futures strategies require leverage, liquidation, funding, volatility, capital loss, and no-profit-guarantee disclosures.

Treasury strategies require governance approval, risk limits, reporting, treasury loss risk, and no-guaranteed-revenue language.

Educational simulators require simulation-not-live-results and educational-purpose disclosures.

## User Confirmation Object

User confirmations should record user reference, strategy ID, disclosure version, no-profit-guarantee confirmation, market risk, API key risk if applicable, leverage risk if applicable, user responsibility, exchange terms, timestamp, and status.

## Prohibited Marketing Claims

Do not claim guaranteed profit, guaranteed daily income, risk-free strategy, passive income without risk, safe futures strategy, cannot lose, AI guarantees profit, backtested means safe, Axodus protects all user funds, or official exchange partnership without confirmation.

## Safe Language

Use terms such as strategy tool, automation support, risk-managed, subject to market risk, no profit guarantee, user-owned account, least privilege API, educational or product access, performance may vary, past results do not guarantee future results, and strategy may be paused or deprecated.

## Review Triggers

Review is required for public strategy pages, BBA campaigns about Trading, performance claims, user-facing bot launches, futures launches, new exchange integrations, API key storage model changes, token-gated strategy access, and treasury strategy reports.
