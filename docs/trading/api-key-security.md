# Trading API Key Security

Status: Draft  
Version: 0.1.0  
Last Updated: 2026-05-16  
Owner: Trading Nucleus

---

## Purpose

This document defines API key safety expectations for trading products.

## Requirements

- Use least privilege.
- Disable withdrawal permissions.
- Use exchange-side restrictions where available.
- Keep keys revocable.
- Never commit secrets to repositories.
- Store secrets only with approved encryption and secret management.
- Document exchange and API risk clearly.

## User Responsibility

Users remain responsible for exchange accounts, permissions, risk limits, and capital exposure.
