## Purpose

DEX users must understand swap and liquidity risks before using advanced features.

## Education Topics

Topics include input and output tokens, quotes, slippage, minimum received, transaction deadline, failed transactions, AMM basics, pool reserves, pool share, fees, liquidity depth, LP tokens, impermanent loss, token verification, official assets, impersonation tokens, scam tokens, approvals, MEV, front-running, sandwich attacks, marketplace checkout, settlement risk, fees, and refunds.

## Access Education Requirements

Basic swaps may require risk disclosure. High-slippage or unverified-token interactions need enhanced warnings. LP positions should recommend impermanent loss education. Treasury or DAO liquidity requires formal review and governance. Advanced DEX trading may require Academy modules or Proof of Knowledge.

## User Confirmation Fields

Confirmations should include user or wallet reference, action type, disclosure version, slippage risk, token listing risk, impermanent loss for LP actions, MEV risk where relevant, and timestamp.
