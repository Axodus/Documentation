# DOC-ADR-009 — Official Repository Structure

## Title

Official Documentation Repository Structure

## Status

PROPOSED

## Context

The repository contains public documentation, operational instructions, semantic knowledge, historical material, and new governance artifacts.

## Problem

Uncontrolled structural growth can obscure document purpose and increase the cost of discovery and governance.

## Alternatives

1. Keep the current structure without additions.
2. Reorganize all existing documentation immediately.
3. Introduce a prospective structure without moving existing files.

## Proposed Decision

Introduce a prospective governance structure while preserving existing paths and VitePress routes.

## Consequences

Legacy and new structures coexist during transition, requiring clear classification and indexing.

## Open Questions

- Which future migrations are valuable?
- Which directories are public or internal?
- How will empty or unused categories be retired?
