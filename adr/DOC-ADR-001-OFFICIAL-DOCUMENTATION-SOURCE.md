# DOC-ADR-001 — Official Documentation Source

## Title

Official Documentation Source

## Status

PROPOSED

## Context

Axodus documentation exists across public documentation, repository-local governance material, nucleus repositories, and global unversioned instructions.

## Problem

The ecosystem needs an auditable source of documentary authority without erasing domain ownership or treating file location as approval.

## Alternatives

1. Each nucleus is independently authoritative.
2. Global unversioned instructions are authoritative.
3. A single central repository owns all documentation.
4. A versioned central registry coordinates locally owned domain sources.

## Proposed Decision

Adopt a hybrid model with a versioned central governance registry and publication surface while nucleus owners retain authority over their domain facts.

## Consequences

Cross-source conflicts require explicit resolution, and central records must link to domain evidence rather than silently replace it.

## Open Questions

- Which approvals establish the central registry as official?
- Which domain facts must remain local?
- How will existing global instructions transition?
