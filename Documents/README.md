# Axodus Orientation Documents

This package contains strategic, economic, product, trading and agent-orientation documents for the Axodus ecosystem.

## Recommended placement

Use this structure inside the repository or project root:

```txt
/opt/Axodus/
├── docs/
│   ├── strategy/
│   ├── commercial/
│   ├── product/
│   ├── trading/
│   └── agents/
└── .instructions/
    └── coder/
```

## Rule of thumb

- `docs/` stores canonical project knowledge.
- `.instructions/` stores executable guidance for agents and coders.
- Keep strategic documents stable and versioned.
- Keep agent instructions short, operational and task-oriented.
