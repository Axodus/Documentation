## Purpose

This folder contains planning materials, drafts, notes, research references, and historical documents used during the reconstruction of Axodus documentation.

Not every file in this folder represents final or active documentation. For canonical documentation, use the root files and `/docs`.

## Current Reference Folders

- `agents/`: ACS and agent alignment references.
- `commercial/`: commercial and BBA strategy references.
- `product/`: product onboarding and user journey references.
- `strategy/`: governance, constitution, treasury, capital allocation, and economic model references.
- `trading/`: trading principles and doctrine references.
- `archive/`: obsolete or superseded documents.
- `planning/`: active planning notes.
- `research/`: research notes and references.

## Status Categories

- Planning
- Draft
- Historical Reference
- Needs Review
- Superseded

## Usage Rules

- Do not treat old drafts as canonical without review.
- Extract useful context carefully.
- Move obsolete materials to `/Documents/archive` when appropriate.
- Keep canonical documentation in `/docs` or root-level files.
- Do not copy financial, token, trading, or governance claims into canonical docs unless they pass current guardrails.
