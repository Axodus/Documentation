# Documentation Templates

## Purpose

Provides reusable starting points for governed documentation.

## Expected Document Types

Templates for decisions, requirements, specifications, policies, standards, APIs, operations, references, reports, research, releases, proposals, and meetings.

## Ownership

Expected ownership belongs to the Documentation governance maintainer.

## Framework Relationship

Templates provide placeholders only. They do not implement schema validation, generation, approval, or publication.

## Display Status Guidance

Public status badges are resolved from governed metadata and frozen page
classifications. Template authors must not add ad hoc badge markup or infer a
status from prose, repository location, implementation code, or maturity level.
New public pages must provide enough metadata for the centralized resolver or
receive a governed page-level classification before publication.
